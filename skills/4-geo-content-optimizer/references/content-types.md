# Content Type Optimization Guides

## Article / Blog Post

### Primary Goal
Establish topical authority and provide comprehensive information that AI can cite as a source.

### Key Elements

1. **Title (H1)**
   - Include primary keyword/entity
   - Clear indication of content type
   - 50-60 characters optimal

2. **Lead Paragraph**
   - Direct answer to implied question
   - Entity mentions (brand, topic, key terms)
   - Promise of what reader will learn

3. **Body Structure**
   - H2 every 300-500 words
   - H3 for subsections
   - Lists for related items
   - Tables for comparisons

4. **FAQ Section** (Required)
   - Minimum 3 questions
   - Match actual search queries
   - Comprehensive answers

5. **Conclusion**
   - Summary of key points
   - Link to related content
   - Clear next step

### Template Structure

```markdown
# [Topic]: [Specific Angle]

[Direct answer sentence]. [Context/background sentence]. [Promise of value].

## What is [Topic]?

**[Topic]**: [Clear definition in one sentence].

[Expansion with examples and context]

## Why [Topic] Matters

- [Benefit 1 with data]
- [Benefit 2 with data]
- [Benefit 3 with data]

## How [Topic] Works

[Step-by-step explanation or process overview]

## [Topic] vs [Alternative]

| Aspect | [Topic] | [Alternative] |
|--------|---------|---------------|
| [Factor 1] | [Value] | [Value] |
| [Factor 2] | [Value] | [Value] |

## Frequently Asked Questions

**Q: [Common question 1]?**

A: [Comprehensive answer with data]

**Q: [Common question 2]?**

A: [Comprehensive answer with data]

**Q: [Common question 3]?**

A: [Comprehensive answer with data]

## Key Takeaways

- [Point 1]
- [Point 2]
- [Point 3]

[Next steps or related resources]
```

### Checklist
- [ ] Title includes primary entity
- [ ] First paragraph answers core question
- [ ] Key terms defined on first use
- [ ] At least one data point per 300 words
- [ ] 3+ FAQ items
- [ ] Headers used throughout
- [ ] Lists for 3+ related items
- [ ] Brand mentioned in first 100 words

## Product Page

### Primary Goal
Provide clear, specific product information that AI can cite in purchase recommendations.

### Key Elements

1. **Product Name (H1)**
   - Clear, specific name
   - Version/variant if applicable

2. **One-Sentence Description**
   - What it is
   - Who it's for
   - Key benefit

3. **Key Specifications**
   - Technical details
   - Dimensions/capacity
   - Compatibility

4. **Use Cases**
   - Who should use this
   - Specific scenarios
   - Success stories/data

5. **Comparison**
   - Vs. alternatives
   - Vs. other products in line
   - When to choose this

### Template Structure

```markdown
# [Product Name]

[Product category] for [target audience] that [key benefit].

## Overview

[2-3 paragraphs covering:
- What the product does
- Key differentiators
- Ideal customer profile]

| [Spec 2] | [Value] |
| [Spec 3] | [Value] |

### [Feature 2]
[Description with benefit and data if available]

### [Feature 3]
[Description with benefit and data if available]

## Use Cases

**[Use Case 1]**: [Specific scenario and outcome]

**[Use Case 2]**: [Specific scenario and outcome]

**[Use Case 3]**: [Specific scenario and outcome]

## Comparison

### [Product] vs [Competitor/Alternative]
```

## FAQ Page

### Primary Goal
Answer common questions comprehensively so AI can quote answers directly.

### Key Elements

1. **Page Title**
   - "[Topic] FAQ" or "Frequently Asked Questions"
   - Clear scope indication

2. **Questions**
   - Match actual user queries
   - Group by topic if 10+
   - Order by importance/frequency

3. **Answers**
   - Complete and standalone
   - Include data/sources
   - Link to detailed resources

### Template Structure

```markdown
# [Topic] FAQ

Common questions about [topic] answered by [Brand] experts.

## General Questions

**Q: What is [Topic]?**

A: [Clear, comprehensive definition]

**Q: How does [Topic] work?**

A: [Step-by-step or conceptual explanation]

**Q: Why is [Topic] important?**

A: [Explanation with data and context]

## [Category 1] Questions

**Q: [Specific question]?**

A: [Detailed answer]

**Q: [Specific question]?**

A: [Detailed answer]

## [Category 2] Questions

**Q: [Specific question]?**

A: [Detailed answer]

**Q: [Specific question]?**

A: [Detailed answer]

## Getting Help

**Q: Where can I learn more about [Topic]?**

A: [Links to resources]

**Q: How do I contact support?**

A: [Contact information]
```

### Answer Best Practices

1. **Complete Sentences**: Answer should make sense alone
2. **Specific Data**: Include numbers, dates, statistics
3. **Entity Mentions**: Brand, product, people names
4. **Source Citations**: "According to [Source]..."
5. **Related Links**: Deep-dive resources
6. **Length**: 50-150 words ideal

### Checklist
- [ ] Questions match real user queries
- [ ] Answers standalone (no context needed)
- [ ] Each answer has specific data
- [ ] Brand mentioned in 50%+ answers
- [ ] 5-15 questions (not too few, not overwhelming)
- [ ] Organized by category if 10+

## Landing Page

### Primary Goal
Convert visitors while providing enough context for AI to understand and cite value proposition.

### Key Elements

1. **Headline (H1)**
   - Clear value proposition
   - Primary benefit
   - Target audience signal

2. **Subheadline**
   - Expand on headline
   - Key differentiator
   - Supporting evidence

3. **Benefits Section**
   - 3-5 key benefits
   - Each with evidence
   - Visual if possible

4. **Social Proof**
   - Customer logos
   - Testimonials with names
   - Usage statistics

5. **Clear CTA**
   - Single primary action
   - Low-friction alternative

### Template Structure

```markdown
# [Value Proposition for Target Audience]

[Subheadline expanding on value with differentiator]

## What You Get

### [Benefit 1]
[Description with specific outcome and data]

### [Benefit 2]
[Description with specific outcome and data]

### [Benefit 3]
[Description with specific outcome and data]

## How It Works

1. **[Step 1]**: [Brief description]
2. **[Step 2]**: [Brief description]
3. **[Step 3]**: [Brief description]

## Trusted By

[Customer logos/names]

> "[Testimonial quote]" — [Name, Title, Company]

## Key Features

| Feature | Description | Benefit |
|---------|-------------|---------|
| [Feature 1] | [What it does] | [Why it matters] |
| [Feature 2] | [What it does] | [Why it matters] |
| [Feature 3] | [What it does] | [Why it matters] |

## Frequently Asked Questions

**Q: What makes [Product] different from [Alternative]?**

A: [Specific differentiator with data]

**Q: How quickly can I get started?**

A: [Timeline with setup steps]

**Q: What kind of support do you offer?**

A: [Support options with response times]

## Get Started

[Primary CTA]

[Secondary CTA]
```

### Checklist
- [ ] Headline states clear value
- [ ] Target audience obvious
- [ ] 3+ benefits with evidence
- [ ] Social proof with names/companies
- [ ] 3+ FAQ items addressing objections
- [ ] Clear single CTA
- [ ] Brand mentioned in first 50 words

## About Page

### Primary Goal
Establish brand entity and authority for AI systems.

### Key Elements

1. **Company Name (H1)**
   - Clear, official name
   - Tagline if applicable

2. **One-Sentence Description**
   - What you do
   - Who you serve
   - Year founded

3. **Origin Story**
   - Why founded
   - Founders
   - Key milestones

4. **Mission/Values**
   - What you stand for
   - How you're different
   - Proof points

5. **Credentials**
   - Awards
   - Notable customers
   - Media mentions

### Template Structure

```markdown
# About [Company Name]

> [One-sentence description including founding year]

## Our Story

[Origin story with:
- Founders and backgrounds
- Problem that sparked creation
- Key milestones with dates
- Growth trajectory with data]

## What We Do

[Company] provides [solution] for [audience]. 

[Key capabilities with specific features]

[Differentiators with proof]

## Our Mission

[Clear mission statement]

### Our Values

**[Value 1]**: [Explanation with example]

**[Value 2]**: [Explanation with example]

**[Value 3]**: [Explanation with example]

## Recognition

- [Award 1, year]
- [Award 2, year]
- [Media mention 1]
- [Media mention 2]

## Our Customers

Trusted by [number] companies including [notable names].

> "[Customer testimonial]" — [Name, Company]

## Leadership

### [Founder/CEO Name], [Title]
[1-2 sentence bio with relevant credentials]

### [Executive Name], [Title]
[1-2 sentence bio with relevant credentials]

## Contact

[Contact information and locations]
```

### Checklist
- [ ] Full company name in H1
- [ ] Founding date mentioned
- [ ] Founders named
- [ ] What you do (clear)
- [ ] Who you serve (specific)
- [ ] Mission stated
- [ ] Proof points (awards, customers)
- [ ] Contact information
- [ ] Brand mentioned 5+ times naturally
